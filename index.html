Projects

LangCollect

LangCollect is a full stack language learning application that leverages interactive chat and flashcards to help users acquire proficiency in a foreign language. The application employs the ChatGPT API, which dynamically adjusts its level of communication based on the user's vocabulary. Additionally, the app integrates the Google Translate package to facilitate real-time translation of single words. With LangCollect, users can immerse themselves in the language learning process and make steady progress towards their fluency goals.

Spots

Airbnb Clone

Spots is a full stack airbnb clone, with a focus on easy image uploads and location-based search. The application utilizes Amazon Web Services (AWS) to facilitate seamless image uploads, storage, and retrieval, ensuring that images are delivered quickly and reliably. Additionally, Spots integrates the Google Maps API to enable users to search for rental listings based on location and to view listings on an interactive map. With Spots, users can easily find and book unique and affordable accommodations for their travels.

Writer's Block

Writer's Block is a web application that allows users to write and share short stories easily. Users can easily collaborate and receive feedback from other writers. It's the perfect tool for unleashing creativity and overcoming writer's block.
