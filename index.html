LangCollect site

GitHub

LangCollect is a full-stack language learning application that combines interactive chat and flashcards to assist users in achieving proficiency in a foreign language. Developed using React, Redux, and Express, this platform leverages the ChatGPT API to dynamically adjust its level of communication based on the user's vocabulary and language proficiency. Additionally, LangCollect integrates the Google Translate package, enabling real-time translation of single words within the chat interface.

Spotsbnb site

 GitHub

Spotsbnb is an Airbnb clone built with React, Redux, and Express. It allows users to book and host homes, with convenient image upload using AWS.

Writer's Block site

 GitHub

Writer's Block is a user-friendly web application that helps writers overcome creative obstacles. Developed using React, Redux, Python, and Flask, it integrates Chat GPT to provide users with an effortless way to write and share short stories.
